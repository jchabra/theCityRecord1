About

The City Record prototype website demonstrates a working prototype of The City Record using five reels of data from 1873-1876. Users are able to log in and search by keyword. The advanced search option allows users to search by date, agency, and location keyword. Search results can be viewed as a downloadable PDF file.

The prototype search functionality is powered by a database (created in MongoDB) populated with OCR metadata extracted from scanned Tiff files. The Tiff files used for this prototype are test files. The prototype will help the project team understand their specific OCR metadata needs and serve as a basis for user testing.
